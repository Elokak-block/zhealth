# **App Name**: Vitality Compass

## Core Features:

- Question Engine: A dynamic engine delivering 30-35 lifestyle-related questions via sliders/multiple choice.
- Lifestyle Strain Index (LSI): Calculates a Lifestyle Strain Index (0-100) based on weighted health pillars to assess overall lifestyle strain. Generates flags when results pattern-match disease risk profiles, offering insights on contributing habits.
- Risk Signal Flags: Flags indicate increased risk alignment based on lifestyle patterns, providing explanations and behavior change suggestions.
- Personalized Result Tiers: Presents results within psychologically safe tiers (Low, Moderate, High, Critical) with clear explanations of primary risk drivers and actionable changes.
- Community Connection Tool: AI-driven tool connects users with similar lifestyle patterns, offering a space for private discussion and shared resources to encourage tool adoption and further reflection.
- Shareable Results: Creates shareable, image-based results summaries. Presents only overall score and tier (Low, Moderate, High, Critical). Includes a QR code with tracking disabled, for loading full report on the application. Result summary clearly features legal disclaimers about non-diagnosis and need for qualified medical opinion.

## Style Guidelines:

- Primary color: Soft teal (#72C5B7), representing health and balance.
- Background color: Light beige (#F5F5DC) to create a calming, neutral base.
- Accent color: Muted orange (#D6896A) to highlight key information and calls to action.
- Body and headline font: 'PT Sans' (sans-serif) for clear and accessible readability.
- Simple, clean icons representing each health pillar (metabolic, inflammation, etc.).
- One-page layout with clear progress indicator.
- Subtle animations for transitions and feedback.
